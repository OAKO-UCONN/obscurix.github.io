Anonymity Networks

Obscurix comes with support for Tor, I2P and Freenet with browsers configured for each. All network traffic is forced through Tor with an exception for I2P and Freenet.

I2P and Freenet are not enabled or installed by default. The installers are in your home directory and Obscurix comes with easy scripts to use them. After installation, they are enabled. The installers are verified with GPG.

I2P

I2P is a peer-to-peer, anonymous network made to protect against surveillance and censorship. Unlike Tor, I2P does not have an officially supported way to access the clearnet.

To install I2P, run installi2p, install it into /home/freenet/i2p and click the I2P Browser desktop icon.

Freenet

Freenet is a peer-to-peer, anonymous, distributed data store.

To install Freenet, run installfreenet, install it into /home/freenet/Freenet and click the Freenet Browser desktop icon.

I2P and Freenet Browsers

The I2P and Freenet browsers that are installed by default, are copies of the Tor Browser that is configured especially for those networks.

Chaining Anonymous Networks

Obscurix does not chain anonymous networks. For example, it prevents I2P from being routed over Tor. This is because chaining anonymous networks may worsen anonymity by making you stand out from other users of those networks. Chaining these networks will also greatly decrease speed and may cause other issues.

Non-Anonymous Networks

Obscurix includes more than just anonymity networks. It also includes many decentralised and encrypted networks. These networks are all forced through Tor to preserve your anonymity.

Zeronet

Zeronet is a peer to peer, decentralised network using bittorrent and bitcoin cryptography with built-in Tor support.

It can be accessed via the Zeronet browser. Click the desktop icon to start it.

IPFS

IPFS is a peer to peer protocol and network designed for security and speed.

It was not designed for privacy or anonymity and there are many ways to track and fingerprint an IPFS user such as this attack. As such, it is not enabled by default.

If you boot with the ipfs boot parameter, it will be enabled. See Usage for more details.

cjdns

cjdns is an encrypted IPv6 network. As it uses IPv6 and IPv6 is disabled in Obscurix, it is not enabled by default. It is installed by default though and users who wish to use it can re-enable IPv6 and enable it.

To enable it, you will need to remove the ipv6.disable=1 boot parameter and add the cjdns boot parameter. See Usage for more details.
