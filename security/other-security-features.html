Other Security Features

Other security features that do not fit in any of the other pages are documented here.

Umask

Umasks control the default file permissions of newly created files. The default umask in Obscurix is set to 0077 in /etc/profile.d/umask.sh. This makes new files only readable by the owner.

Mirrors

Pacman is configured to only use HTTPS mirrors. This prevents malicious exit nodes from injecting code during updates and it prevents exit nodes from seeing what you're downloading.

Thunar

Thunar is configured so it uses the ISO format for dates, disables thumbnails, disables thunar-volman and disables the network bookmark.

Entropy

Jitterentropy and haveged is installed by default to gather more entropy.
